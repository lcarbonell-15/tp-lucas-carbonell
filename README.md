# Impacto de la Ley 27.642 sobre General Cereals S.A.

Sitio web interactivo del TP Grupal — Laboratorio de Metodos Cuantitativos, FCE UBA, 1C 2026.

## Deploy en Streamlit Community Cloud

1. Crear un repositorio en GitHub (publico o privado)
2. Subir todos los archivos de esta carpeta al repositorio:
   - `app.py`
   - `requirements.txt`
   - `Dataset_Limpio.csv`
   - `.streamlit/config.toml`
   - `README.md`
3. Ir a [share.streamlit.io](https://share.streamlit.io)
4. Conectar tu cuenta de GitHub
5. Seleccionar el repositorio y el archivo `app.py`
6. Click en **Deploy**
7. Esperar ~2 minutos a que se instalen las dependencias
8. Listo — tenes tu URL publica para compartir

## Ejecucion local

```bash
pip install -r requirements.txt
streamlit run app.py
```
